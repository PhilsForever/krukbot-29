## KrukBot29

A robot version of John Kruk that lives in the r/Phillies subreddit. Not affiliated with John Kruk or the Phillies organization in any way.

## What Am I?

> A bot that replies to some comments in game threads where someone mentions John Kruk

1. Randomly chooses a phrase from a list to reply to comments.
2. Has an array of text phrases that provides a specific reply for those phrases.
3. Creates a leaderboard of Kruk mentions to see who mentions Johnny the most.
4. Only replies to comments 25% of the time so it isn't annoying.
5. Responds to "!krukbot-leaderboard" to show top Kruk mentions.
